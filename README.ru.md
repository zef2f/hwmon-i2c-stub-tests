# hwmon i2c-stub тесты

Набор тестов для hwmon-драйверов ядра Linux. В качестве программной
шины I2C используется модуль `i2c-stub` - реальное оборудование не нужно.

## Требования

Конфигурация ядра:

```
CONFIG_HWMON=y
CONFIG_I2C=y
CONFIG_I2C_STUB=m
CONFIG_I2C_CHARDEV=m
```

А также модуль тестируемого драйвера (например, `CONFIG_SENSORS_LM90=m`).

Пакеты: `i2c-tools` (`i2cset`, `i2cget`).

## Запуск

```
$ sudo ./run-tests.sh [драйвер ...]
```

Без аргументов запускает все драйверы из `list.txt`. Требуются права root.

Статусы тестов:

- `[Ok]` - успешно
- `[Failed]` - тест завершился с ошибкой
- `[Skip]` - модуль драйвера недоступен
